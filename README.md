📽️ Presentador de Diapositivas Interactivo

Este proyecto consiste en un presentador de diapositivas dinámico en HTML5, CSS y JavaScript, diseñado para mostrar imágenes y videos con transiciones animadas, navegación interactiva y diseño responsivo.

✨ Características principales
🎞️ Presentación de imágenes y videos
⬅️➡️ Navegación con botones anterior y siguiente
⌨️ Control con teclado (← y →)
🔘 Indicadores de diapositivas
🎨 Transiciones aleatorias entre slides
📱 Diseño adaptable a diferentes pantallas
🌙 Fondo oscuro elegante tipo presentación profesional
🦶 Footer personalizado
🛠️ Tecnologías utilizadas
HTML5
CSS3
JavaScript ES6 (Modules)
🎬 Funcionalidades
Navegación manual

El usuario puede desplazarse entre diapositivas mediante botones laterales.

← Diapositiva anterior
→ Siguiente diapositiva

También puede usar el teclado:

ArrowLeft
ArrowRight
🎨 Efectos de transición

El sistema incluye múltiples animaciones dinámicas:

Fade
Slide Left
Slide Right
Slide Up
Slide Down
Rotate
Scale
Flip X
Flip Y
Zoom Blur
Swing In
Bounce In
Glitch
Spiral In
Drop In
Unfold

Cada transición se aplica aleatoriamente al cambiar de diapositiva.
